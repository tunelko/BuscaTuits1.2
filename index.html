BuscaTuits (LungoJS 1.2 version)

Another twitter search webapp with pulldown to refresh ;-). LungoJS is a fantastic pure HTML5 framework to work with and oriented to mobile devices. This webapp search for an input topic and lists tweets with 'pulldown to refresh' on LungoJS v1.2.

LungoJS modifications:

  • Implemented prepend method to iScroll.
